# Online Quiz System

The Online Quiz System is a web-based application designed to conduct quizzes and automatically evaluate user performance. It allows students to attempt quizzes and receive instant results while enabling teachers to manage quiz questions easily.

## Features
- Student login and quiz participation
- Teacher panel to manage questions
- Multiple choice questions
- Automatic score calculation
- Simple and user-friendly interface

## Technologies Used
- Python
- Django
- HTML
- CSS
- JavaScript
- SQLite / MySQL

## Project Modules
- Student Module – Allows students to take quizzes and view results
- Teacher Module – Enables teachers to create and manage quiz questions
- Quiz Module – Handles quiz logic and scoring
- Templates – Contains HTML pages
- Static – Includes CSS and JavaScript files

## How to Run the Project
1. Install Python on your system
2. Install dependencies using `requirements.txt`
3. Run the command `python manage.py runserver`
4. Open the browser and go to `http://127.0.0.1:8000/`
